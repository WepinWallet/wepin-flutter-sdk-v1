# wepin_flutter_network

[![pub package](https://img.shields.io/pub/v/wepin_flutter_network.svg?logo=flutter&style=for-the-badge)](https://pub.dartlang.org/packages/wepin_flutter_network)

Wepin Network SDK for Flutter. This package is exclusively available for use in Android and iOS environments.

## ⏩ Install
Add the `wepin_flutter_network` dependency in your pubspec.yaml file:

```yaml
dependencies:
  wepin_flutter_network: ^0.0.1
```
or run the following command:

```bash
flutter pub add wepin_flutter_network
```
